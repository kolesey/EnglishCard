# Телегам бот Englesh Cards
## Описание Функционала
Данный телеграм бот предоставляет возможность сохранять изучаемые слова в базе данных.
Для сохранения можно предложить свое слово или выбрать случайно предложенное.
Перевод слов осуществляется через сервисы Яндекс Переводчика.

## Настройка бота
Для запуска необходимо переименовать файл .env.example в .env и заполнить необходимые параметры.
